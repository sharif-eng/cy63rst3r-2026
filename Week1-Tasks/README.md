# 🔴 Week 1 - Advanced Reconnaissance & Attack Surface Mapping

## 🎯 Objective
Master information gathering and identify the digital footprint of a target without detection.

## 📝 Tasks
1. 🔍 **Passive OSINT & Subdomain Enumeration**  
   - Tools: Subfinder, Assetfinder, Amass, DNSDumpster, CRT.sh  
   - Goal: Cross-reference results to identify hidden assets.  

2. 🏗️ **Infrastructure & Technology Profiling**  
   - Tools: WhatWeb, Wappalyzer, Google Dorking, WHOIS, NSLookup  
   - Goal: Map tech stack to narrow down exploits.  

3. 🔑 **GitHub Dorking & Credential Leaks**  
   - Tools: GitHacker, GitHub search queries  
   - Goal: Identify leaked secrets and credentials.  

4. 🌐 **Visual Reconnaissance & Live Asset Identification**  
   - Tools: Httprobe/Httpx, Aquatone, Gowitness  
   - Goal: Focus only on active, high-value targets.  

## 📑 Weekly Deliverable
Submit a **Reconnaissance Report** containing:
1. 📝 Executive Summary  
2. 📊 Asset List (CSV/table of subdomains & IPs)  
3. 🛠️ Technology Map  
4. ⚠️ Security Gaps (low-hanging fruit)
